Privacy Policy

Effective date: March 19, 2026

Local List Assist is designed around local-first software. The Android app works in free local mode by default, and the Home Assistant version stays local-only. Optional paid sync is available for Android users who want backup, multi-device access, and shared spaces.

Android app data in local mode

The Android app can be used in local mode without a subscription.

  • List names, colors, categories, and ordering are stored locally on device.
  • Items, completed history, archived lists, and recent activity are stored locally on device.
  • Local backups created through the app are stored in app-managed device storage.
  • The current version does not include ads or behavior-tracking analytics.

Android optional sync

If you sign in and enable paid sync, Local List Assist processes the data needed to provide personal sync spaces, shared spaces, multi-device access, and entitlement verification.

Website data

The website is a simple public marketing and policy site. If standard web hosting logs are collected by the hosting provider, they are limited to ordinary operational data such as IP address, request path, and user agent for delivery and security purposes.

Data sharing

Local List Assist does not sell personal data. The app does not use third-party advertising identifiers or behavior-tracking analytics in the current version.

Backups

If you export a local backup from the Android app, the resulting file is stored on the device. You control whether it is retained, copied elsewhere, or deleted.

Home Assistant version

The free Home Assistant integration runs inside Home Assistant and stores its data in the user’s HA environment. That version is separate from the standalone Android app and follows the behavior of the user’s Home Assistant installation and hosting setup.

Future changes

If future versions add optional features or change how data is processed, this policy will be updated to reflect those changes before broad rollout.

Contact

Questions about privacy or support can be sent to support@locallistassist.app.
