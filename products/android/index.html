Android app

Free local lists first. Paid sync only if you need it.

Local List Assist for Android is the standalone version of the product. It works in free local mode by default, so you can manage lists on one device without paying first. If you want backup, multi-device access, or shared spaces with other people, optional paid sync is available inside the app.

Included in the Android app

  • Multiple lists
  • Grocery and flat list templates
  • Category-aware grocery grouping
  • Duplicate handling and review
  • Completed history
  • Archived lists
  • Recent activity
  • Local backup and restore
  • Optional shared sync spaces

Sync model

Sync is optional. A subscribed account gets one personal sync space automatically, and that same user can still join other sync spaces as a member. The app should remain fully useful even if you never enable sync.

Product stance

This is utility software. No ads. No behavior-tracking analytics. No Home Assistant requirement. Local mode stays straightforward, and sync is there only for the people who actually need it.

Availability

The Android release is in testing and staged rollout work. If you want launch access or support, email support@locallistassist.app.
