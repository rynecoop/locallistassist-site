Free Home Assistant version

Use Local List Assist inside Home Assistant.

This version is for people who already run Home Assistant and want local-first list management directly inside HA. It keeps the product idea intact while fitting the Home Assistant workflow instead of forcing a separate cloud service or subscription.

What it is good at

  • Native Home Assistant panel experience
  • Smart grocery routing and category learning
  • Multiple internal lists
  • Category editing and ordering
  • Fast local refresh across open clients
  • A free option for existing HA users

See it in action

This short desktop walkthrough shows the current Local List Assist panel running inside Home Assistant, including quick add, multiple lists, category sections, and the completed area.

Who should use it

Use the Home Assistant version if Home Assistant is already the place where your household automations, dashboards, and voice flows live. Use the Android app if you want a standalone product with optional sync and no Home Assistant dependency.

Cost

The Home Assistant version is free and local-only. The standalone Android app is a separate product with free local mode and optional paid sync.
